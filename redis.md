# redis

## 安装rides

`sudo pacman -S redis` :使用 pacman 包管理器安装 rides

## 启动redis

`systemctl enable redis` :让 rides 开机自启

`systemctl start redis` :启动 rides

`redis-cli` :会让我进入这个终端(redis 127.0.0.1:6379>)
